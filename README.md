# Med Bot: Clinic Automation Assistant

## Overview
An agentic AI assistant designed for AuraCare Clinic to streamline patient management. The bot automates appointment scheduling and handles patient inquiries by interacting directly with a backend database.

## Key Features
* **Agentic Workflow:** Built using Langflow to manage complex patient queries.
* **Database Integration:** Real-time tool-calling to sync appointments with the clinic's MySQL database.
* **Professional Guardrails:** Implemented security protocols to ensure accurate and professional clinical responses.

## Tech Stack
* **AI/Workflow:** Langflow
* **Database:** MySQL
* **Backend:** Node.js, Express
* **Frontend:** HTML, CSS
